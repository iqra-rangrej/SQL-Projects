📊 Data Digger – SQL Project

📌 Overview

Data Digger is a SQL-based project that simulates an E-Commerce database. It helps in understanding database management using CRUD operations, SQL queries, and relationships.

---

🎯 Objective

- Perform CRUD operations
- Use SQL clauses (WHERE, ORDER BY, BETWEEN)
- Apply aggregate functions (SUM, AVG, MAX, MIN, COUNT)
- Understand primary & foreign keys

---

🗂️ Tables Used

1. Customers
2. Orders
3. Products
4. OrderDetails

---

⚙️ Features

- Insert, update, delete, and retrieve data
- Filter and sort records
- Calculate total revenue and averages
- Identify top and least values

---

🚀 Tools

- MySQL
- SQL

---

📌 Conclusion

This project builds a basic understanding of relational databases and real-world SQL usage in E-Commerce systems.

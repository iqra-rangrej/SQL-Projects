📌 Project Overview

The Data Transformer SQL Project demonstrates the use of SQL to manage and analyze structured data.
It works with multiple datasets including customers, orders, and employees to perform data transformations and generate meaningful outputs.

---

🗂️ Data Used

- Customers – customer information and registration details
- Orders – order transactions and amounts
- Employees – employee records and salary data

---

⚙️ SQL Operations Performed

🔹 Joins

- INNER JOIN
- LEFT JOIN
- RIGHT JOIN

🔹 Subqueries

- Filtering records using nested queries
- Comparing values with averages

🔹 Window Functions

- Running Total
- Ranking using RANK()

🔹 String Functions

- CONCAT
- UPPER / LOWER
- TRIM

🔹 Date Functions

- Extract year and month
- Format dates
- Calculate date differences

🔹 Conditional Logic

- CASE statements
- Discount calculation

---

🚀 How to Run

1. Create tables in MySQL
2. Insert data
3. Execute queries

---

🛠️ Tools Used

- MySQL 8.0
- MySQL Workbench
